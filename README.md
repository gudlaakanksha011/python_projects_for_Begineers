# Python Programming Portfolio

A collection of functional Python scripts demonstrating fundamental programming concepts, including control flow, data manipulation, and user input handling.

## 🛠 Projects Included
1. **Password Generator**: Creates secure random strings.
2. **To-Do List**: A task management utility.
3. **Simple Calculator**: Basic math operations.
4. **Word Count**: Text analysis tool.
5. **Number Guessing**: Interactive logic game.
6. **Rock Paper Scissors**: Classic game vs AI.

## 🚀 How to Run
Clone the repo and run any file using Python:
`python filename.py`

## 📈 Skills Shown
* Loops and Conditionals
* String Manipulation
* Randomization Modules
* User Input Handling
